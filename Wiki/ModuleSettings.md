## Module Settings

The module settings will give you (and your players) a lot of performance options as well as some other stuff

- Selection Highlight image: This image will replace the default ugly squares that are used if no image is provided, this image will show whenever a token is selected or highlighted. I suggest using any of the turn marker indicators (for example from Combat Booster) animated assets are not supported

- Stand-ups face camera: When using tokens in standup mode (aka a 2d image that stands up) this optio will determine if those tokens always face the camera

- Prevent negative elevations: if enabled will prevent drag movement to go below the grid when changing elevation

- Enable Canvas Popout: If enabled will automatically open the 2d canvas in a separate window when the 3d mode is engaged. Keep in mind that you can always open\toggle the canvas popout with the button under the cube button to enable the 3d mode

- Disable light/shadows: This is a perormance option that will massively boost framerate on lower end hardware - all dynamic lighting will be disabled and only a global light will be left

- Resolution: High (full resolution), Medium (half resolution), Low (quarter resolution)
