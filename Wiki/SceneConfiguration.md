## Scene Configuration

You will get a massive amount of settings but do not worry :)

 - Enable 3D on load: Automatically load the 3d mode when loading the scene

 - Skybox image: The image to be used as skybox, check resources(3d).md for links to where to get them already formatted with the correct names

 - Display Table: Enabling this will create a "table" in the area where the padding of your 2d scene is, if you don't set a texture it will be black - i suggest wood textures from forgotten adventures

 - Table Texture: The texture for the table, this texture will be stretched not tiled so use something with a decent resolution

 - Show Scene Walls: Display walls setup with wallheight on the 3d scene (this is the same as the old levels3dpreview view)

 - Show Scene Floors/Polygons: Same as above but tiles

 - Render scene lights: Create foundry lights in the 3d environment as well, keep in mind that for performance reasons they will illuminate through objects
